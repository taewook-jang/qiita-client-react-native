qiita-client-react-native
========================

React Native Sample Qiita Client App

Features

- Sign in Qiita
- Showing posts in ListView
- Showing user profile

## Installation

```
npm install
```

### Run on ios

```
react-native run-ios
```

## Dependencies

- react-redux
- redux-logger
- redux-thunk
- immutable
- moment
- react-native-router-flux
- etc ...
